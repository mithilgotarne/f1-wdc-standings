# f1-wdc-standings
Dynamic Video Generator based on WDC standings


## Steps:

- clone repo
- run `pip install -r requirements.txt`
- run `python3 demo.py`
- output will be generated as `out.mp4`

## Usage:

- run after every race weekend :)
